# Apex CRM - Enterprise Sales Intelligence

A premium, full-stack CRM system built for high-performance sales management.

## Tech Stack
- **Frontend**: React, Vite, Framer Motion, Recharts, Lucide Icons, Vanilla CSS
- **Backend**: Node.js, Express, MongoDB, Socket.io, JWT

## Getting Started

### Prerequisites
- Node.js installed
- MongoDB installed and running locally on port 27017

### Installation

1. **Backend Setup**:
   ```bash
   cd backend
   npm install
   npm run dev
   ```

2. **Frontend Setup**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## Features
- **Lead Tracking**: Advanced repository with real-time updates and status tracking.
- **Sales Analytics**: Industrial-grade dashboard with revenue projections and pipeline status.
- **RBAC**: Integrated Role-Based Access Control (Admin, Manager, User).
- **Silent Luxury UI**: Dark-themed, glassmorphic interface designed for clarity and focus.

## Project Structure
- `/backend`: API services, Models, and Auth controllers.
- `/frontend`: React client with modular components and shared state.
